  <action>
    1. Add weasyprint and pikepdf to project dependencies:
       ```
       uv add weasyprint pikepdf
       ```

    2. Create `bookforge/build/pdf.py`:
       ```python
       def build_pdf(
           html_string: str,
           output_path: Path,
           fmt: str,                    # "screen" | "print"
           trim_inches: tuple[float, float],
           bleed_inches: float,
           book_dir: Path,
       ) -> Path:
       ```

       Implementation:
       - Create `FontConfiguration()` instance
       - Load the font-face CSS string with absolute path to bundled NotoSansKR-Regular.otf
       - Create CSS objects passing `font_config=font_config` to each
       - Call `HTML(string=html_string, base_url=str(book_dir)).write_pdf(output_path, stylesheets=css_objects, font_config=font_config, optimize_images=True)`
       - CRITICAL: `font_config` must be passed to BOTH `CSS()` and `write_pdf()` — per research, passing to only one silently disables @font-face
       - If fmt == "print", call `patch_print_pdf(output_path, trim_inches, bleed_inches)`
       - Return output_path

    3. Create `bookforge/build/postprocess.py`:
       ```python
       def patch_print_pdf(
           pdf_path: Path,
           trim_inches: tuple[float, float],
           bleed_inches: float,
       ) -> None:
       ```

       Implementation (per research Pattern 5):
       - Convert inches to PDF points (72 pt/in)
       - For 8.5x8.5 trim + 0.125 bleed: trim_w_pt=612, trim_h_pt=612, bleed_pt=9
       - Note: For square books, both width and height get bleed on all 4 edges, so MediaBox = [0, 0, trim_w_pt + 2*bleed_pt, trim_h_pt + 2*bleed_pt]
       - Open with `pikepdf.open(pdf_path, allow_overwriting_input=True)`
       - Set MediaBox, TrimBox, BleedBox on every page:
         - MediaBox = [0, 0, media_w, media_h]
         - TrimBox = [bleed_pt, bleed_pt, media_w - bleed_pt, media_h - bleed_pt]
         - BleedBox = MediaBox (bleed extends to media edge)
       - Embed sRGB ICC OutputIntent:
         - Read `bookforge/assets/icc/sRGB_v4_ICC_preference.icc`
         - Create pikepdf.Stream with N=3, Alternate=/DeviceRGB
         - Create OutputIntent Dictionary with Type=/OutputIntent, S=/GTS_PDFA1, OutputConditionIdentifier="sRGB IEC61966-2.1"
         - Use `pdf.make_indirect(output_intent)` before appending (per research Pitfall 4)
         - Append to pdf.Root["/OutputIntents"]
       - Save with `linearize=True`

    4. Create `tests/test_pdf_export.py`:
       - Test build_pdf creates a file (use a minimal HTML string, not the full template)
       - Test patch_print_pdf box dimensions with pikepdf assertions (create a tiny PDF with WeasyPrint, patch it, re-open and assert TrimBox values)
       - Test ICC OutputIntent is present after patching
       - Test screen format does not have TrimBox set
       - Use a helper to create minimal test PDFs (simple HTML with WeasyPrint) to avoid needing full book fixtures

    IMPORTANT anti-patterns to avoid:
    - Do NOT use `local()` for font loading — always `url()` with absolute path
    - Do NOT add `marks: crop` — KDP rejects crop marks; use `marks: none`
    - Do NOT hardcode dimensions — compute from trim_inches and bleed_inches parameters

       ```python
       def build_command(
           slug: str = typer.Argument(..., help="Book slug (directory name under books/)"),
           lang: str = typer.Option("bilingual", "--lang", "-l",
                                    help="Edition: en, ko, bilingual, all"),
           fmt: str = typer.Option("all", "--format", "-f",
                                   help="Format: screen, print, all"),
       ) -> None:
       ```

       Implementation:
       - Resolve book_dir = Path("books") / slug; exit with error if not found
       - Load book via `parse_story(book_dir / "story.md")`
       - Load style_guide via `load_style_guide()` (same pattern as illustrate.py)

       - Create output_dir = book_dir / "output"; mkdir -p
       - For each (edition, format) combination:
         - html = render_book_html(book, edition, style_guide, format, book_dir)
         - output_path = output_dir / f"{slug}-{edition}-{format}.pdf"
         - build_pdf(html, output_path, format, trim_inches, bleed_inches, book_dir)
