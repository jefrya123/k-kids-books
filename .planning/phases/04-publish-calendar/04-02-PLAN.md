    - compute_deadlines(entry) returns dict with: release_date (21 days before holiday), marketing_start (35 days before holiday = 14 days before release), writing_start (56 days before holiday = 8 weeks), illustration_start (42 days before holiday = 6 weeks)

  <action>
    Create bookforge/calendar.py with:
    1. `CalendarEntry` as a Pydantic BaseModel with fields: holiday_name (str), holiday_date (date), slug (str), status (str = "planned")
    2. `load_calendar(path: Path) -> list[CalendarEntry]` -- reads YAML with ruamel.yaml, validates each entry, returns sorted by holiday_date. YAML format is a list of dicts: `- holiday_name: "Chuseok"\n  holiday_date: 2026-09-25\n  slug: chuseok-story\n  status: planned`

    4. `get_upcoming(entries: list[CalendarEntry]) -> list[CalendarEntry]` -- filters to entries where holiday_date >= today

    - Test FileNotFoundError on missing YAML
