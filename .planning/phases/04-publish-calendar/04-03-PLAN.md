  <files>bookforge/publish/__init__.py, bookforge/publish/covers.py, tests/test_publish.py</files>
  <behavior>
    - compute_spine_width(page_count) returns page_count * 0.002252 (inches, white paper)
    - compute_kdp_cover_dimensions(trim_w, trim_h, spine_width) returns (total_width_inches, total_height_inches) where total_width = 2*trim_w + spine + 2*0.125 (bleed each side), total_height = trim_h + 2*0.125
    - generate_gumroad_thumb(cover_image_path, output_path) resizes cover to 1600x2560 PNG
    - generate_social_square(cover_image_path, output_path) resizes/crops cover to 1080x1080 PNG
    - generate_kdp_cover(cover_image_path, trim_w, trim_h, page_count, output_path) creates a full KDP cover spread: back cover (trim_w) + spine (computed) + front cover (trim_w), with 0.125" bleed on all edges, at 300 DPI. Front cover is the provided cover image; back is solid color from image dominant color; spine is narrow strip with title text rotated 90deg
    - All image operations use Pillow
    - Edge: 12-page book spine = 12 * 0.002252 = 0.027024" (very thin -- spine text may not fit, just use colored strip)
  </behavior>
  <action>
    Create bookforge/publish/__init__.py (empty).

    Create bookforge/publish/covers.py with:
    1. `compute_spine_width(page_count: int) -> float` -- returns page_count * 0.002252

    3. `generate_gumroad_thumb(cover_path: Path, output_path: Path) -> Path` -- open with Pillow, resize to 1600x2560 (LANCZOS), save PNG
    4. `generate_social_square(cover_path: Path, output_path: Path) -> Path` -- open, crop center square, resize to 1080x1080, save PNG
    5. `generate_kdp_cover(cover_path: Path, trim_w: float, trim_h: float, page_count: int, output_path: Path, dpi: int = 300) -> Path` -- compute spine, compute total dimensions in pixels (inches * dpi), create new RGB image, paste front cover on right, fill back and spine with dominant color extracted via image.resize((1,1)).getpixel((0,0)), save PNG. If spine >= 0.5" (222+ pages), draw rotated title text on spine; otherwise leave as solid color.
